Detailed Description

Loads light field data from Light Field Picture files (Lytro's raw.lfp files).

Utilites Nirav Patel's lfdsplitter (https://github.com/nrpatel/lfptools) to split the file into it's components and rapidjson (https://code.google.com/p/rapidjson/) to extract image metadata.
